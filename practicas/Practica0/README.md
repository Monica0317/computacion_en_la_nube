# Práctica 0 – Preparación del entorno

## Objetivo
Configurar el entorno de trabajo Vagrant + VirtualBox + Ubuntu que se
usará en el curso y aprender como se publican boxes en Vagrant Cloud.

## Actividades realizadas
- Instalación del sistema operativo Linux
- Instalación de Git
- Configuración básica de Git
- Creación del repositorio en GitHub
- Estructura inicial de directorios.
